How to Use the Map Method

From Ruby-doc.org

map { |obj| block } --> array

Returns a new array with the results of running block once for every element in enum.

In other words, .map is a lot like .each although you can add a bang (!) to the end of .map which will make it a destructive method. It will then change the original array. Honestly, I'm not sure exactly how the regular .map is different than .each. This will be something for further exploration in the future.
