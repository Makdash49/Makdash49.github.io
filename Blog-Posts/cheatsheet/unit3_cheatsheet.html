Basic Document Setup

 There is a basic structure for HTML documents. Here it is:

<!DOCTYPE html>- This defines the document type.

<html>- This describes the website.

<head>- This is the part of the code where you put the title and include links to other documents.

<title>This is an Example of a Title</title>

</head>You close the head with this tag.

<body>- This is where you will put the main content and most of your code.

</body>- And you close the body like this.

</html>- Finally, you close the html with this tag.

Using images in HTML

The Image tag allows you to add images to your document.

<img src="someURL"> Will add the image at someURL to your page.

HTML Anchor Tags

The Anchor tag lets you jump between sections of your document, and also make links to other web pages or resources, including email addresses.

<a href="SomeURL"> Your browser will display this text for the user to click on to go to the section </a>
