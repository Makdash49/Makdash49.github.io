JavaScript: Strengths and Weaknesses

by Mark McQuillen

July 1, 2014

What's So Great About JavaScript?

JavaScript is popular in web development because it can help develop software quickly while easily creating a great user experience. This makes it very useful in creating applications for mobile devices. With the development of Node.js it can now participate in server-side development. Many high school students enjoy learning JavaScript because it's quick and easy to learn so its popularity is expected to stay strong.

Additional Great things about JavaScript:

  1. JavaScript saves bandwidth when it is executed on the client side.
  2. Uses the DOM model for simplicity and clarity.
  3. Javascript gets to the end user fairly fast.

What are the Limitations of JavaScript?

JavaScript has some limitations which include a security issue. Javascript snippets can be used to exploit a user's system. Also Javascript rending varies depending on the layout engines which can result in inconsistency in terms of functionality and interface.

Additional JavaScript Limitations:

  1. JavaScript cannot write files on the server without help from server side script.
  2. Javascript cannot access databases in many cases
  3. Javascript cannot write or read files in the client.
